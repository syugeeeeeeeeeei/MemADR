# CHG-012: Linux向けインストーラスクリプトを配布物へ追加した

Status: SHIPPED
Reason: GitHub Release から Linux 利用者が `curl ... | sh` 形式で `memadr` バイナリを導入できるようにし、README 冒頭から最短で導線を辿れるようにするため
Change: `buildctl dist` が `memadr-install.sh` を配布物へ含めるようにし、同内容の `scripts/memadr-install.sh` をリポジトリへ置いた。スクリプト内では Linux amd64 向け release バイナリの取得、導入先の決定、バージョン固定、repo 上書きを扱えるようにした。あわせて README 冒頭へ Linux インストール案内を追加し、配布物一覧へ installer asset を反映した
FilesChanged: README.md, cmd/buildctl/main.go, internal/release/install.go, internal/release/install_test.go, scripts/memadr-install.sh, memory/changes/CHG-012.md
Verification: `go test ./...`、`go run ./cmd/buildctl dist -version v0.0.1 -out-dir <temp>`
