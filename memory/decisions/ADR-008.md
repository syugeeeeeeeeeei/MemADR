# ADR-008: initでワークフロー案内を同梱する

Status: ACCEPTED
Context: `help` だけではコマンド仕様は見えても、人間やLLMが実際にどの順で `memadr` を使うかの導線が分かりにくい
Decision: `memadr init` は標準ディレクトリに加えてリポジトリ直下へワークフロー案内Markdownを生成し、`help` はそのファイル名へ誘導する
Rejected: `help` へ長い運用説明を全て埋め込む案、外部ドキュメントだけに依存する案
Consequence: 初回導入時点で人間とLLMの共通参照先を揃えられ、CLIヘルプは短く保ったまま運用導線を示せる
