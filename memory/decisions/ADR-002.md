# ADR-002: 開発タスクランナーをPowerShell依存Makefileからjustfileへ移行する

Status: ACCEPTED
Context: 既存の `Makefile` は `powershell.exe` と PowerShell 構文に強く依存しており、Windows 以外で同じ開発フローを再利用しにくい
Decision: 開発タスクランナーを `justfile` に置き換え、主要レシピは `go` コマンドを直接呼び出し、OS差分はシェル選択とバイナリ名に限定する
Rejected: Makefileを維持したままシェル分岐を増やす案
Consequence: `just` の導入は必要になるが、Windows/macOS/Linuxで同名レシピをほぼ同じ操作感で利用できる
