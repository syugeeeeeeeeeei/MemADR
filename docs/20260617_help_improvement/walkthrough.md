# 実施内容

## 変更概要

- `memadr` 引数なし実行で総合ヘルプを表示するようにした
- `memadr help [command]` で個別ヘルプを出せるようにした
- コマンド定義からヘルプを描画する構成へ寄せた
- レコード種別の説明を `mem` 側に持たせ、`new` のヘルプへ再利用した

## 確認方針

- `go test ./...` でヘルプ出力と既存 `init` `new` 動作を確認する
- `go run ./cmd/memadr` で総合ヘルプを目視確認する

## 補足

今後コマンドを増やすときは、コマンド定義に説明と使用例を追加すれば、総合ヘルプと個別ヘルプの両方に反映される。
